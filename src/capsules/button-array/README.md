# Button Array Capsule

CSS-only capsule. Registers `.button-group` and `.button-group__item` styles for pill-connected button rows.

## Usage

```html
<drop capsule="button-array"></drop>
```

## Features

- Horizontal pill-connected layout, stacks vertically on small screens
- Hover expands focused item, shrinks siblings
- Supports first/middle/last item radius variants

## Notes

Pages must use `.button-group` / `.button-group__item` classes on their own markup.
