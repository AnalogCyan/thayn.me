# Card List Capsule

CSS-only capsule. Registers `.card-list` and `.card-list__item` styles for stacked glass cards.

## Usage

```html
<drop capsule="card-list"></drop>
```

## Features

- Vertically stacked glass cards with outer corner rounding
- Hover expands focused card, shrinks others
- Pseudo-element glass highlight and edge glow
- Dense variant for blog cards (`.card-list--dense`)

## Notes

Pages must use `.card-list` / `.card-list__item` classes on their own markup.
